# Vutureland

A modular Hyprland desktop configuration for Arch Linux.

![[./assets/icons/vuturland.png]]

---

## Features

- **Hyprland** — Wayland compositor (Lua-based config via `hypr.lua/`)
- **Waybar** — Modular status bar with interactive setup wizard
- **Wallpaper theming** — `awww` + `wallust` auto-generate color schemes from wallpapers
- **Submaps** — Window management, navigation, quickstart, and developer keybind layers
- **Floating window cascading** — Smart cascade + clamp for floating windows
- **Per-monitor bars** — Each monitor gets its own Waybar config built by `waybar.sh`
- **Bluetooth notifications** — systemd service sends desktop notifications on connect/disconnect
- **Lock screen** — `hyprlock` with workspace save/restore and playback pause
- **Session menu** — `wlogout` for logout/lock/sleep/reboot/shutdown
- **Rofi wallpaper picker** — Browse wallpapers with thumbnails via Rofi

## Setup

### 1. Hyprland

Run the interactive configurator to set up your device-specific settings:

```bash
bash ~/.config/vutureland/.setup/hyprland.sh
```

This generates `hypr/user_settings.conf` with your monitors, cursor, keybindings, and startup apps.

### 2. Waybar

Run the interactive setup to build a Waybar config for each monitor:

```bash
bash ~/.config/vutureland/.setup/waybar.sh
```

Select monitor → position → style → modules for each slot (left / center / right), then save.
To rebuild after changes:

```bash
bash ~/.config/vutureland/.setup/waybar.sh --rebuild
```

### 3. Wallpaper

Set a wallpaper and auto-generate colors:

```bash
bash ~/.config/vutureland/assets/scripts/wallpaper-set.sh ~/path/to/wallpaper.jpg
```

Or pick one via Rofi (mapped to a keybind in the quickstart submap).

### 4. Settings GUI

```bash
python3 ~/.config/vutureland/gui/main.py
```

Provides a graphical interface for monitors, Waybar slot editing (drag-and-drop), wallpaper browser, and color mode toggle.

## Waybar Modules

Each module lives in `waybar-modular/modules/horizontal/<name>/` with:
- `config.json` — Waybar module config
- `style.css` — Module styling
- `module.md` — Metadata (name, description, alias, slot restriction)

To restrict a module to a specific bar slot, add to its `module.md`:
```
specific = "left"   # or "center" / "right"
```

## Bluetooth Notifications

Enable the bt-notify systemd service for connection/disconnection alerts:

```bash
# Copy the service file (see header of assets/scripts/bt-notify.sh)
systemctl --user daemon-reload
systemctl --user enable --now bt-notify.service
```
